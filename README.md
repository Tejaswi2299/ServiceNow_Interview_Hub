# ServiceNow Interview Hub

Static GitHub Pages package for ServiceNow interview preparation.

## Included
- roles, modules, and topics navigation
- coding questions
- use case scenarios
- quiz
- bookmarks
- Google Analytics 4 support CTA integration

## Publish
1. Upload the folder contents to your repository.
2. In GitHub **Settings → Pages**, choose **Deploy from a branch**.
3. Select your branch and the root folder.
4. Save and wait for GitHub Pages to publish.

## Notes
- This package is static HTML, CSS, JavaScript, and JSON.
- No local install is required.
- Content is designed to keep expanding over time.
